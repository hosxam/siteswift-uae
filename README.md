# SiteSwift UAE — Business System

## Overview

This is a complete, ready-to-launch business system for a semi-automated AI website agency targeting UAE small businesses (clinics, gyms, cafes, real estate, etc.).

**Revenue model:** Monthly subscription (AED 499-1,299/month per client)
**Target clients:** Small-to-medium UAE businesses with outdated or no websites
**Your edge:** AI-powered site generation + medical/tech background = unique positioning
**Startup cost:** $0 (all tools have free tiers)

## Getting Started in 30 Minutes

### Step 1: Set Up Your Presence
1. Upload `demo-website-template.html` to GitHub Pages as your landing page
2. Customize the WhatsApp number and email in the template
3. Set up a professional email (name@yourbusiness.com or use Gmail)

### Step 2: Set Up Tools
1. Create accounts on: Brevo (email sequences), Stripe (billing), UptimeRobot (monitoring)
2. Open `dashboard.html` in your browser for lead management
3. Add `lead-database-template.csv` to Google Sheets for easy editing

### Step 3: Find Your First 5 Clients
1. Open Google Maps → search "clinics in Dubai Marina" → find 20 leads
2. Add them to your CSV
3. Email 5 per day using the outreach templates
4. Follow up on day 3 and day 7

### Step 4: Build Demos
1. When a lead responds, use the `demo-website-template.html` as a base
2. Customize their branding, copy, and info
3. Deploy a preview link via GitHub Pages
4. Send it and close the deal
